

Cosas a aplicar para hacer un buen testing


Python - PyTest y Unittest


Para ejecutar los tests:

- Crea una carpeta con los archivos.
- Crea el algoritmo a testear.
- Importa los módulos a testear en el archivo de testing.
- Abre el CMD.
- Ubícate en la carpeta.
- Ejecuta pytest o pytest -v.
- Ejecuta un archivo específico con pytest -s nombre.py.
- Para ver el tiempo de ejecución: pytest --durations=0 -vv.
- Para ejecutarlo con Unittest: pon python -m unittest nombre.py.
