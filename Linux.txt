

Comandos para manejar la GUI de Kali Linux:


Terminal


Ctrl + Shift + T: Nueva pestaña.

Ctrl + Shift + W: Cerrar pestaña.

Ctrl + Shift + N: Nueva ventana.

Ctrl + +: Ampliar.

Ctrl + -: Reducir.

Ctrl + 0: Reiniciar el zoom.


Mousepad - Editor de texto


Ctrl + N: Nuevo archivo.

Ctrl + W: Eliminar archivo.

Ctrl + S: Guardar archivo.

Ctrl + O: Abrir archivo.

Ctrl + Shift + S: Guardar archivo como ....

Ctrl + Shift + W: Cerrar ventana.

Ctrl + Q: Salir.

Ctrl + A: Seleccionar todo.

Ctrl + F: Buscar una palabra.


Universal:


Ctrl + Shift + C: Copiar.

Ctrl + Shift + V: Pegar.
