

Comandos básicos


Windows: abre o cierra el menú de inicio.

Windows + A: abre el panel de acciones.

Windows + B: activa los íconos en la barra de tareas.

Windows + D: esconde las apps abiertas.

Windows + ALT + D: muestra el calendario.

Windows + E: abre el explorador de archivos.

Windows + G: abre la barra de juegos.

Windows + I: abre la configuración.

Windows + K: conexión rápida.

Windows + L: bloquea la pantalla.

Windows + M: minimiza todas las ventanas.

Windows + Shift + M: restaura las ventanas minimizadas.

Windows + P: modo de proyección de la pantalla.

Windows + R: ejecutar comandos de Windows.

Windows + S: abrir búsqueda.

Windows + Shift + S: captura de pantalla o parte de ella.

Windows + T: recorrer aplicaciones en la barra de tareas.

Windows + V: historial del portapapeles.

Windows + .: abre el panel de emojis.

Windows + 1-9: abre aplicaciones de la barra de tareas.

Windows + Tab: vista de aplicaciones abiertas.

Windows + Flechas: acomoda ventanas.

Windows + Inicio: minimiza todas las ventanas excepto la actual.

Windows + Pausa: abre las propiedades del sistema.

Windows + Ctrl + D: crea un escritorio virtual nuevo.

Windows + Ctrl + F4: cierra el escritorio virtual actual.

Windows + Shift + Left/Right Arrow: mueve la ventana activa a otro monitor en una configuración de varios monitores.

Windows + Ctrl + F: busca equipos en una red local.

Windows + H: comparte contenido a través de la función "Near Share".

Windows + X: abre el menú de usuario avanzado.

Windows + U: abre el Centro de accesibilidad.

Windows + Shift + Símbolo de suma (+): aumenta el zoom de la pantalla.

Windows + - (guion): reduce el zoom de la pantalla.

ALT + F4: cierra la app de Chrome.
